Tahukah kamu?

Menu

Indonesia terdiri atas berbagai macam suku bangsa dan budaya yang berbeda-beda, namun tetap dalam satu wadah Negara Kesatuan Republik Indonesia (NKRI). Keberagaman tersebut merupakan anugerah dari Tuhan Yang Maha Esa sehingga kita wajib mensyukurinya. Kita tidak boleh merendahkan suku bangsa lain dan menganggap suku bangsa sendiri sebagai suku bangsa yang terbaik.

Step-step

Menu

Instalasi Bahan

  • Sebelum melakukan konfigurasi multisite CMS WordPress, terlebih dahulu kita harus melakukan installasi Xampp yang digunakan sebagai server localhost.
  • Setelah selesai installasi Xampp, selanjutnya adalah melakukan installasi CMS WordPress dengan hanya copy paste file CMS WordPress ke folder xampp/htdocs.

Konfigurasi

Setelah langkah installasi aplikasi yang dibutuhkan selesai dilakukan, maka langkah selanjutnya untuk memanfaatkan fitur multisite pada CMS WordPress untuk membangun sebuah subdomain atau subdirectory. Berikut langkah-langkah mengkonfigurasi fitur multisite pada CMS WordPress.

  • Buka file wp-config.php yang terdapat pada directory CMS WordPress.
  • Setelah file wp-config.php sudah terbuka, maka tambahkan sytag beriktu "('WP_ALLOW_MULTISITE',true);" dan simpan.
  • Setelah selesai menambahkan sytag pada file wp-config.php, selanjutnya adalah buka web browser pilih menu tools, ketika pada sub menu Tools terdapat menu network setup seperti gambar berikut :
  • Setelah memilih opsi Network Setup pada menu Tools, maka halaman untuk mengkonfigurasi multisite terbuka seperti gambar berikut
  • konfigurasi sesuai dengan kebutuhan, dalam fitur multisite WordPress terdapat 2 metode, yang pertama subdomain dan subdirectory
  • Setelah itu klik install untuk meneerapkan fitur multisite pada CMS WordPress
  • Setelah selesai menginstal fitur multisite, maka akan muncul halaman untuk menambahkan dan mengkonfigurasi pada file-file WordPress seperti gambar berikut.
  • Untuk kolom pertama, copy dan paste syntag tersebut pada file wp-config.php
  • Untuk kolom kedua, copy dan pasti syntag tersebut pada file .htaccess
  • Setelah selesai melakukan copy paste pada file wp-config.php dan .htacess, maka penginstalan multisite WordPress telah sukses dilakukan seperti gambar berikut
  • Untuk membuat halaman subdomain maupun subdirectory dilakukan pada network admin seperti gambar diatas.
  • Demikian tutorial untuk memanfaatkan fitur multisite pada CMS WordPress
